## 📖 О проекте

**Антимошенник** — это веб-приложение-тренажёр, разработанное для повышения цифровой грамотности и защиты от социальной инженерии. Игра предлагает пользователю анализировать диалоги между мошенником и жертвой, выбирая признаки обмана.

### 🎯 Цель

Обучить людей разных возрастов распознавать типичные схемы мошенничества:
- 📱 **Фишинг** — кража логинов и паролей
- 💳 **Банковское мошенничество** — запрос кодов и CVV
- 👨‍👩‍👧 **Социальная инженерия** — звонки от «родственников» в беде
- 🎁 **Ложные выигрыши** — оплата налога за приз
- 🖥️ **Техподдержка** — удалённый доступ к компьютеру

## 🎮 Как играть?

1. **Выберите возраст** — от этого зависит сложность и тематика диалогов
2. **Выберите режим времени**:
   - ⚡ 1 минута — быстрый режим для опытных
   - ⏳ 5 минут — спокойное прохождение
   - ♾️ Бесконечный — без ограничений, только для обучения
3. **Анализируйте диалог** между мошенником (красные облака) и жертвой (синие облака)
4. **Отмечайте признаки мошенничества** из предложенного списка
5. **Нажимайте «ответить»** и получайте очки

### 📊 Система начисления очков

В каждом диалоге разное количество правильных признаков (от 2 до 5+). Чтобы игра была честной, используется **нормированная система** — максимум за раунд **всегда 100 очков**.

Вес одного правильного признака = 100 / (общее число правильных признаков в диалоге)

Очки = (правильные × вес) − (ложные × вес × 0.7) − (пропущенные × вес × 0.35)

| Возраст | Категория | Примеры ситуаций |

## ✨ Особенности

### 🎨 Дизайн
- **Пастельные тона** — красный для мошенника, синий для жертвы
- **Нестандартные облака сообщений** — уникальные стили сообщений
- **Анимированные переходы** — плавное появление и исчезновение диалогов
- **Минималистичный интерфейс** — без лишних элементов, фокус на контенте

### 🎬 Анимации
- ✨ Плавное появление/исчезновение меню (0.3 сек)
- 🌫️ Блюр-эффект между раундами (0.5 сек)
- 📨 Fade-анимация смены диалогов (0.4 сек)
- 🔘 Hover-эффекты на всех интерактивных элементах

### 🛡️ Безопасность
- Запрет выделения текста на всей странице (кроме сообщений)
- Отсутствие копирования элементов интерфейса
- Чистый JavaScript без внешних зависимостей
